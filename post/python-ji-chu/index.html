调用函数

capitalize 使首字母大写，其他字母小写

用法：newstr = string.capitalize()
参数：函数括号内什么都不用写
细节：

name = 'xiaoMu'
new_name = name.capitalize()

.lower() 将字符串全体小写

用法：newstr = string.casefold()
newstr = string.lower()
参数：括号内什么都不写
细节：只对字符串中字母有效、已经是小写则无效；casefold有将其他语言大写转小写的功能，出的更晚，但平时string.lower()就足够用了

.upper() 将字符串全体大写

用法：big_str = string.upper()
参数：函数括号内什么都不用写
细节：只对字符串中字母有效、已经大写则无效
