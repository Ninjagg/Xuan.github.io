混合数据类型的列表中 max/min无法运算

调用函数

capitalize 使首字母大写，其他字母小写

用法：newstr = string.capitalize()
参数：函数括号内什么都不用写
细节：

name = 'xiaoMu'
new_name = name.capitalize()

.lower() 将字符串全体小写

用法：newstr = string.casefold()
newstr = string.lower()
参数：括号内什么都不写
细节：只对字符串中字母有效、已经是小写则无效，字符串内可以有数字、符号等内容不会报错，正常对字母部分转换；casefold有将其他语言大写转小写的功能，出的更晚，但平时string.lower()就足够用了

.upper() 将字符串全体大写

用法：big_str = string.upper()
参数：函数括号内什么都不用写
细节：只对字符串中字母有效、已经大写则无效，字符串内可以有数字、符号等内容不会报错，正常对字母部分转换。

.swapcase() 将字符串大小写转换

用法： newstr = string.swapcase()
参数：函数括号内什么都不写
细节：字符串内可以有数字、符号等内容不会报错，正常对字母部分转换。

.zfill 为字符串定义长度，如不满足，缺少的部分'在字符串前面'用0填补

用法：newstr = string.zfill(width)
参数：width: 新字符串希望的宽度
细节：与字符串的字符无关；如果定义长度小于当前字符串长度，则不发生变化

print() 打印对象

细节：print('逗号' = '一个空格'; '加号' = '没有空格')

A = 'liu'
B = 'xuan'
print(A, B)  # liu xuan
print(A + b)  #  liuxuan

count() 返回当前字符串中某个成员(元素)的个数 (返回一个数字)

用法：inttype = string.count(item)
参数：item: 查询个数的元素
细节：如果查询的成员(元素)不存在，则返回0

startswith() endswith() 判断开头或结尾是否是设定的元素

用法：string.startswith(item)
string.endswith(item)
参数：item：你想查询匹配的元素
细节：返回一个bool值，二者可以对首尾字符串进行判断，也可以对整个字符串进行判断

find() index() 返回你想要元素的位置

用法：string.find(item)
string.index(item)
参数：item：你想查询的元素
细节：返回一个整型；字符串里位置是从左向右，以0开始的；find找不到这个元素会返回-1，index找不到这个元素，会导致程序报错；日常中也许find更适合开发——懒惰匹配（只匹配字符串中该元素第一次出现的位置）

strip()去掉字符串左右两边的指定元素，默认是空格

用法：newstr = string.strip(item)
参数：item:你想去掉的元素，不填即默认'空格'
细节：传入的元素如果不在开头或结尾则无效；无论首尾有多少空格都可以去掉；lstrip()用法与strip()一致但仅能去掉字符串开头的指定元素或空格；rstrip()相同但仅能去掉字符串结尾的指定元素或空格
